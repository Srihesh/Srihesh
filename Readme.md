### Social Hyena

The canonical problem with social graphs at scale is that standard attention is computationally paralyzed by them. Quadratic cost is not an engineering shortcut to work around — it is a fundamental architectural mismatch. Social Hyena replaces the transformer with a Hyena long convolution operator running via Fast Fourier Transforms, compressing millions of noisy user nodes into overlapping semantic supernodes through motif-based tokenization. The representation preserves bridge users that rigid clustering erases. Global network reasoning drops to sub-quadratic complexity and noise filters itself by construction.

`sub-quadratic sequence mixing` &nbsp; `hierarchical motif tokenization` &nbsp; `social graph language modeling`

<br>

### Temporal GNN Pipeline

Cross-dataset generalization across temporal social networks is a quietly unsolved problem. Networks speak different temporal languages — different cadences, different interaction semantics, different structural norms — and the same architecture that excels on one regime fails on another without clear signal as to why. Current work builds context-conditioned convolution kernels that read graph-agnostic structural fingerprints and dynamically adapt their behavior, letting one model reason coherently across incompatible network regimes.

`anomaly detection` &nbsp; `cross-dataset generalization` &nbsp; `graph neural networks`

### RELABTeams

A multi-role collaboration platform for students and recruiters built on the MERN stack. The interesting design challenge was not the features but the trust architecture — role-governed APIs, JWT-based access control, OTP and TOTP two-factor authentication, and approval-gated recruiter workflows all living inside a single unified system. Production integrations include Cloudinary for asset management and Razorpay with server-side signature verification for payments. CI/CD via GitHub Actions, deployed to Vercel.

`MERN` &nbsp; `JWT` &nbsp; `RBAC` &nbsp; `CI/CD` &nbsp; `Razorpay`

<br>

### Aadhaar Pulse

A high-performance data pipeline built to analyze population-level identity update patterns across massive biometric and demographic datasets. The core contribution is the Identity Entropy Index — an algorithm that quantifies administrative volatility and infers migration and life-event signals from raw update logs with high statistical significance. Results rendered as interactive geo-temporal dashboards from state down to pincode level.

`DuckDB` &nbsp; `Pandas` &nbsp; `Plotly` &nbsp; `custom algorithm design`

<br>

### Semicolon

A cyberpunk-themed web platform that makes regex feel intuitive by executing patterns visually, step by step. Built a custom regex parsing algorithm that decomposes expressions into visual components, showing users exactly how and why a pattern matches. Includes a live testing engine and a pattern library for real-time validation. The aesthetic was a deliberate choice — complexity dressed in something that does not look like a textbook.

`JavaScript` &nbsp; `custom parser` &nbsp; `interactive UI`

<br>

### Glitchnetcult

A semantic document retrieval system that pulls unstructured text from PDFs, chunks it intelligently, generates high-dimensional embeddings, and indexes them in a vector database for low-latency semantic search. The architecture lays the foundation for a full RAG pipeline. Containerized as a microservice with Docker for consistent cross-environment deployment.

`Python` &nbsp; `NLP` &nbsp; `vector database` &nbsp; `RAG architecture` &nbsp; `Docker`
